## 5. PROMPT TEMPLATES LIBRARY (EXPANDED)
*Copy, paste, and fill in the bracketed information `[ ]` for your tasks.*

### Category: Marketing & Content Creation
**Use Case:** Generating engaging social media posts or ad copy from boring product specs.

> **Template:**
> "Act as a Senior Social Media Manager.
> **Context:** We are launching a new feature: [Feature Name/Description].
> **Target Audience:** [e.g., Busy professionals, Gen Z, Stay-at-home parents].
> **Platform:** LinkedIn (Professional & Concise) AND Twitter/X (Punchy & casual).
> **Task:** Write 3 options for EACH platform.
> **Constraints:**
> * Include a strong 'Hook' in the first sentence.
> * Include a clear Call to Action (CTA) at the end.
> * Use relevant emojis."

### Category: Sales & Negotiation
**Use Case:** Handling difficult client objections or cold outreach.

> **Template:**
> "Act as a top-performing Sales Consultant.
> **Scenario:** I am selling [Product/Service] to a potential client in the [Industry] industry.
> **The Objection:** The client just said: '[Paste objection, e.g., "Your price is too high compared to Competitor X"].'
> **Task:** Provide 3 distinct responses to handle this objection.
> 1.  **The Empathy Approach:** Validate their concern first.
> 2.  **The Value Approach:** Pivot to ROI and unique benefits.
> 3.  **The Challenger Approach:** Gently push back on the comparison.
> **Tone:** Confident but respectful."

### Category: Coding & Development (Debugging)
**Use Case:** Fixing broken code and understanding *why* it broke.

> **Template:**
> "Act as a Senior Software Engineer.
> **Context:** I have a bug in the following [Language, e.g., Python/JavaScript] code.
> **Task:**
> 1.  Analyze the code step-by-step.
> 2.  Identify the logical error or syntax mistake.
> 3.  Provide the corrected code block.
> 4.  Explain the fix simply so a junior developer can understand.
> **Code Snippet:**
> ```
> [Paste code here]
> ```"

### Category: Strategic Planning (SWOT Analysis)
**Use Case:** Quickly analyzing a competitor or a new project idea.

> **Template:**
> "Act as a Business Strategy Consultant.
> **Subject:** [Company Name or Project Idea].
> **Context:** [Brief description of what the company/project does].
> **Task:** Perform a SWOT Analysis (Strengths, Weaknesses, Opportunities, Threats).
> **Output Format:**
> * Present the result in a Markdown Table.
> * List at least 4 items per quadrant.
> * Prioritize the most critical 'Threat' and suggest a mitigation strategy."

### Category: Learning & Training (Quiz Generation)
**Use Case:** Creating training materials from documentation.

> **Template:**
> "Act as an Instructional Designer.
> **Input Text:** [Paste article, manual, or transcript here].
> **Task:** Create a 5-question multiple-choice quiz based on the text above to test understanding.
> **Output Format:**
> * Question
> * 4 Options (A, B, C, D)
> * Correct Answer
> * Explanation of *why* it is correct."

### Category: Prompt Generator (The "Meta-Prompt")
**Use Case:** When you don't know how to write the prompt, ask the AI to write it for you.

> **Template:**
> "I want you to become my Prompt Engineer.
> **My Goal:** I need to [Describe what you want to do, e.g., 'Analyze customer churn data' or 'Write a mystery novel'].
> **Task:** Write the best possible prompt for me to use with ChatGPT/Gemini to achieve this goal.
> **Requirements:** Include a Persona, Context, Constraints, and Output Format in the prompt you generate for me."
